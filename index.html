1. Überblick

BuroClear ist eine Offline-App, die Einwanderern in Deutschland hilft, offizielle Dokumente zu verstehen. Alle Daten werden ausschließlich auf Ihrem Gerät verarbeitet. Es werden keine Daten an externe Server übertragen.

a) Dokumententext

Der Text wird ausschließlich im Arbeitsspeicher (RAM) verarbeitet, nicht auf der Festplatte gespeichert und nicht an Server gesendet.

b) Kameraaufnahmen

Fotos werden nur zur Texterkennung (OCR) genutzt und sofort nach der Erkennung aus dem Speicher entfernt. Die Kameranutzung erfordert Ihre ausdrückliche Zustimmung.

c) Spracheinstellung

Die gewählte App-Sprache wird lokal in UserDefaults gespeichert. Dies ist die einzige dauerhaft gespeicherte Information.

d) KI-Analyse

Die Analyse erfolgt vollständig lokal über Apple Intelligence (FoundationModels). Kein Text wird an Server gesendet.

  • Keine personenbezogenen Daten
  • Keine Standortdaten
  • Keine Nutzungsstatistiken oder Analytics
  • Keine Werbe-Identifikatoren (IDFA)
  • Keine Cookies oder Tracking
  • Keine Datenübertragung an Dritte
  • Keine Serververbindungen
  • Keine IP-Adressen oder Geräte-Identifikatoren

5. Drittanbieter & Datenübermittlung

Keine Drittanbieter-SDKs, keine Cloud-Dienste, keine Datenübermittlung in Drittländer. Die App funktioniert vollständig offline.

6. Speicherdauer
